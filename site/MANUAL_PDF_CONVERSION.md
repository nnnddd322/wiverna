# Ручная конвертация DOCX в PDF

Автоматические скрипты могут зависать. Рекомендуется сделать конвертацию вручную через Word:

## Способ 1: Через Microsoft Word (рекомендуется)

1. Откройте каждый файл из папки `lec/` в Microsoft Word:
   - `дисциплина №1.docx`
   - `Дисциплина №2.docx`
   - `Дисциплина №3.docx`
   - `Дисциплина №4.docx`
   - `Дисциплина №6.docx` (пропускаем №5)
   - `Дисциплина №7.docx`
   - `Дисциплина  №8.docx`

2. Для каждого файла:
   - Нажмите **Файл → Сохранить как** (или F12)
   - Выберите тип файла: **PDF**
   - Убедитесь, что путь сохранения: `pdfs/`
   - Назовите файл точно как указано:
     - `Дисциплина №1.pdf`
     - `Дисциплина №2.pdf`
     - `Дисциплина №3.pdf`
     - `Дисциплина №4.pdf`
     - `Дисциплина №6.pdf`
     - `Дисциплина №7.pdf`
     - `Дисциплина №8.pdf`
   - Нажмите **Сохранить**

3. Проверьте, что все 7 PDF файлов появились в папке `pdfs/`

## Способ 2: Массовая конвертация через Word

1. Откройте Microsoft Word
2. Файл → Открыть → выберите все DOCX файлы из `lec/` (Ctrl+A в папке)
3. Для каждого документа:
   - Файл → Сохранить как → PDF
   - Сохраните в `pdfs/` с правильным именем

## Способ 3: Через пакетную обработку (если Word поддерживает)

Word 2016+ может конвертировать несколько файлов одновременно:
1. Откройте Word
2. Файл → Открыть
3. Выберите все DOCX файлы
4. Для каждого: Файл → Сохранить как → PDF

## Важно

- Названия файлов должны быть **точно** как указано выше (с пробелами и символами №)
- Дисциплину №5 пропускаем (её нет на сайте)
- Всего должно быть 7 PDF файлов

После конвертации PDF файлы автоматически появятся на сайте!
