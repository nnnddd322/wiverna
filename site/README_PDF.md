# Поддержка PDF файлов

## Что было добавлено:

1. **Интерфейс:**
   - Добавлена кнопка "PDF" в переключатель между презентациями, лекциями и PDF
   - Карточки PDF файлов отображаются с иконкой PDF
   - При клике PDF открывается в новой вкладке браузера

2. **Структура данных:**
   - Добавлено поле `pdfFiles` в каждую дисциплину (№1, 2, 3, 4, 6, 7, 8)
   - PDF файлы должны находиться в папке `pdfs/`

3. **Скрипт конвертации:**
   - `convert_docx_to_pdf.ps1` - автоматически конвертирует DOCX файлы из папки `lec/` в PDF

## Как использовать:

### Вариант 1: Автоматическая конвертация
```powershell
powershell -ExecutionPolicy Bypass -File convert_docx_to_pdf.ps1
```

### Вариант 2: Ручная конвертация
1. Откройте DOCX файлы в Microsoft Word
2. Сохраните как PDF (Файл → Сохранить как → PDF)
3. Поместите PDF файлы в папку `pdfs/`
4. Назовите файлы:
   - `Дисциплина №1.pdf`
   - `Дисциплина №2.pdf`
   - `Дисциплина №3.pdf`
   - `Дисциплина №4.pdf`
   - `Дисциплина №6.pdf` (пропускаем №5)
   - `Дисциплина №7.pdf`
   - `Дисциплина №8.pdf`

## Структура файлов:

```
proba/
├── lec/                    # Исходные DOCX файлы
│   ├── дисциплина №1.docx
│   ├── Дисциплина №2.docx
│   └── ...
├── pdfs/                   # PDF файлы (создается автоматически)
│   ├── Дисциплина №1.pdf
│   ├── Дисциплина №2.pdf
│   └── ...
└── data.js                 # Данные с полем pdfFiles
```

## Примечания:

- Дисциплина №5 пропускается (её еще нет на сайте)
- PDF файлы открываются в новой вкладке браузера
- Если PDF файл не найден, браузер покажет ошибку 404
