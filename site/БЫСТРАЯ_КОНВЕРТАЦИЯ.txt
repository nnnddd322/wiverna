  КОНВЕРТАЦИЯ DOCX В PDF - ПРОСТОЙ СПОСОБ
═══════════════════════════════════════════════════════════

Автоматические скрипты зависают. Используйте ручную конвертацию:

1. Откройте Microsoft Word

2. Откройте файлы из папки lec/ (по одному или все сразу):
   - дисциплина №1.docx
   - Дисциплина №2.docx  
   - Дисциплина №3.docx
   - Дисциплина №4.docx
   - Дисциплина №6.docx  (пропускаем №5!)
   - Дисциплина №7.docx
   - Дисциплина  №8.docx

3. Для каждого файла:
   → Нажмите F12 (или Файл → Сохранить как)
   → Выберите тип: PDF
   → Папка: proba/pdfs/
   → Имя файла (ВАЖНО - точно как указано):
      • Дисциплина №1.pdf
      • Дисциплина №2.pdf
      • Дисциплина №3.pdf
      • Дисциплина №4.pdf
      • Дисциплина №6.pdf
      • Дисциплина №7.pdf
      • Дисциплина №8.pdf

4. Нажмите Сохранить

5. Проверьте папку pdfs/ - должно быть 7 PDF файлов

После этого PDF файлы появятся на сайте автоматически!
